# ТЗ:

# 1.Перевод времени
Программа получает на ввод целое число (количество часов),

выводит это количество часов, но переведённое в минуты...

Пример:
входные данные: 2

выходные данные: 120

# 2. Сравнение чисел
Программа получает на ввод целых 2 числа

сравнивает, какое из этих чисел больше,

затем выводит текст, где говориться какое число больше...

Пример:
входные данные: 2 4

выходные данные: Число 4 больше

3.Допуск к экзамену
Программа получает на ввод целое число от 1 до 5 (оценку).

Проверяет оценку на проходной бал (больше либо равно 3)

И выводит строку "допущен" если число проходит И строку "не допущен" если не подходит...

Пример:
входные данные: 3

выходные данные: допущен

# 4.Скидка в магазине
Программа получает на ввод два целых числа (цену товара и количество),

затем подсчитывает общую сумму покупки, если общая цена покупки больше 1000,

то на общую сумму покупки применится скидка в 10%, в остальных случаях цена остаётся прежней.

В конце программа выводит сумму с учётом суммы.

Пример:
входные данные:

200

6

выходные данные: 1080

# 5. Решение квадратного уравнения
Программа получает на ввод 3 числа (a b c)

Подсчитывает их Дискриминант (D) по формуле D=b^2-4ac

если дискриминант меньше нуля, то программа выводит сообщение "корней нет" если дискриминант равен нулю, то программа считает 1 корень (x1) по формуле b^2 / 2a

если дискриминант больше нуля, то программа считает 2 корня (x1, x2) по формуле -b +- sqrt(D) / 2a
